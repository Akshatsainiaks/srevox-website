## ✨ Features

* **Premium Design System**: Dark-mode SaaS branding powered by CSS grids and modern fonts (Plus Jakarta Sans).
* **Interactive Dashboard Simulator**: High-fidelity frontend mockup that displays live crash loop diagnostics, container logs, and simulated AI diagnostic traces.
* **Alert Target Previewer**: Responsive panels to show formatting schemas for Slack, MS Teams, Twilio WhatsApp, and HTML Emails.
* **Environment Builder**: An interactive client-side configurator to generate and copy `.env` files for self-hosting setups.
* **IDE-Style Documentation Navigation**: Custom sidebars displaying detailed setup manuals, RBAC configurations, API references, and simulation tools.
* **Statically Optimized**: Statically generated Next.js compilation for sub-second page loads.

## 🛠️ Local Development

Follow these steps to run the Srevox website locally for editing pages or checking guides:

### Prerequisites
* **Node.js** 18 or higher.
* **npm** (comes pre-bundled with Node.js).

### Getting Started

**1️⃣ Install node dependencies:**
```bash
npm install
```

**2️⃣ Start the development server:**
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the website.

**3️⃣ Compile verification (TypeScript check):**
```bash
npx tsc --noEmit
```

**4️⃣ Production build output:**
```bash
npm run build
```
